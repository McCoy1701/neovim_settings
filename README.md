# My Neovim settings

## Install
1. Create a nvim directory in ~/.config/
2. Copy repo to that newly created directory ~/.config/nvim/
3. Open neovim run Lazy install every thing
4. Restart neovim
5. Open neovim run Mason
6. Repeat step 4
7. Should be done!
